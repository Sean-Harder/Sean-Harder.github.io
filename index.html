2022

Light API Project

January 2022

This project began as a way to learn how to code, starting with using LIFX's API to control lights from my desktop. What began as a simple experiment (Messing_Around.py) in Visual Studio, where I aimed to turn on a light with a few lines of code, soon developed into something more.

Iteration 1

In the first main iteration, I integrated a Weather API, alongside the old light control code, and explored adding the Spotify API. The project dynamically changes the background and images based on weather conditions (snow, rain, sunny), while displaying information like temperature, location, humidity, and air pressure.

Iteration 2

I continued to build and improve on this, refining the user interface with smaller light control options in the bottom right (color and on/off buttons), and organizing other controls in the bottom left. I initially planned to connect it to the Spotify API, but ran into issues with authentication through Tkinter. Though I wasn't able to complete that part at the time, I kept researching and learning with different projects.
